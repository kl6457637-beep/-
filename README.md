# 项目名称：亚马逊广告优化LLM Agent工作流
## 作品体验——ACOS KILLERS (https://code.coze.cn/p/7610719352387731508/preview)
## 项目定位
面向亚马逊跨境卖家的全自动广告优化AI产品**，解决广告ACOS高、人工优化效率低、无效流量浪费的核心痛点，实现广告优化全流程自动化。
（产品经理视角：明确目标用户+核心痛点+产品价值）

## 核心能力
1. **双分支Agent架构**：流量清洗止血 + 关键词收割拓词，覆盖广告优化全链路9个核心节点；
2. **精准Prompt工程**：完成语义判断、场景拓词模块的Prompt设计，保障大模型输出准确率；
3. **全流程自动化**：无需人工介入，单店广告优化从2小时/天压缩至5分钟内。

## 产品实现方案（AI产品经理核心工作）
### 1. 产品架构设计
- 核心逻辑：双分支并行Agent架构（附架构流程图👇）
- 节点定义：明确9个核心节点的功能边界、输入输出与交互规范；
### 2. 技术落地路径
- 工作流搭建：基于Coze完成AI Agent节点配置、逻辑调试；
- 代码实现：通过AI编程工具（如OpenCode）实现功能落地，打通流程闭环；
- 模型适配：完成LLM Prompt调优，适配亚马逊广告场景的语义理解。

## 核心成果（数据化呈现）
- 效率提升：单店广告优化效率提升90%+（2小时/天 → 5分钟/次）；
- 业务价值：帮助卖家降低广告ACOS 15%-30%；
- 可复用性：沉淀的Agent工作流模板可快速适配其他跨境平台广告优化场景。
